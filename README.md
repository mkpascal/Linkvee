# Linkvee

Description
-------------------------

This is a simple web platform to share links to the world. Moreover, it includes different features,

* Thumbnails of the website
* Posting area with reCaptcha
* Reporting interface (bad links)

Attribution
-------------------------

This project uses different tools,

* Kohana 3.0
* WebsnapR
* and reCaptcha.

The front-end design has been made by AG. Famfamfam icons are used.
